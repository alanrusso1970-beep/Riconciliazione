# Guida Rapida di Installazione

## Passo 1: Prerequisiti
- Python 3.7+
- Node.js 16+

## Passo 2: Setup
1. Apri un terminale nella cartella del progetto
2. Esegui: `npm install`
3. Apri un altro terminale ed esegui: `python server.py`
4. In un terzo terminale, esegui: `npm run dev`

## Passo 3: Utilizzo
- Apri il browser su `http://localhost:5173`
- Il server proxy è su `http://localhost:8787`

## Note
- Assicurati che Google Apps Script sia accessibile
- Le porte possono essere modificate se già in uso
